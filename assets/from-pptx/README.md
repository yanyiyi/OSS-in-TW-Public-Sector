# PPTX Extracted Media

This folder contains media extracted directly from the source PowerPoint files.

- `en/ppt/media/`: extracted from `EN.pptx`
- `zh-tw/ppt/media/`: extracted from `ZH_TW.pptx`

The manuals reference selected PNG/JPEG images from these folders. Decorative slide backgrounds, logos, and small icons are also preserved here for completeness.

Note: `hdphoto1.wdp` is JPEG-XR media from the PPTX package. macOS `sips` could not convert it to PNG in this environment, so the original `.wdp` files are preserved as extracted.
