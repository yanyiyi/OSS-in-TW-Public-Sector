# Assets

Images in this folder are extracted from the source PPTX files and renamed with language prefixes.

- `zh-tw-*`: extracted from `ZH_TW.pptx`
- `en-*`: extracted from `EN.pptx`

The Markdown manuals reference selected PNG/JPEG images directly from this single `assets/` folder. `*.wdp` files are JPEG-XR media preserved from the PPTX package.
